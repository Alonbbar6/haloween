A fun Halloween-themed web game where you can choose between a trick or a treat!

## Features

- **Trick**: Watch a fun surprise video
- **Treat**: Play a knock-knock game to win virtual candy
- Interactive animations and sound effects
- Responsive design that works on desktop and mobile

## How to Play

1. Open `index.html` in your web browser
2. Click "TRICK" for a surprise or "TREAT" to play a game
3. In the treat game, knock 3 times quickly, pause, then knock 2 more times to win candy!

## Technical Details

- Built with vanilla HTML, CSS, and JavaScript
- No external dependencies required
- Uses CSS animations for smooth transitions
- Responsive design using flexbox and viewport units

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `script.js` - Game logic and interactivity

## Browser Support

Works in all modern browsers including Chrome, Firefox, Safari, and Edge.
